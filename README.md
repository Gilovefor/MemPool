# MemoryPool
*** 
## 实现内容：
* **分级内存池**：非线性 size-class，Slot槽大小从8~512字节
* **Block大小自适应**：大小增长策略匹配Slot增长策略，减少大池碎片
* **内存对齐**：分配内存按cache line对齐
* **线程局部空闲链表**：每线程维护本地Slot链表，优先复用，降低锁竞争
* **全局无锁链表**：超出阈值后，Slot归还到全局原子链表，实现无所分配与回收

## 核心策略
* **对象分级**：通过二分查找，快速定位合适内存池，提升分配效率。
* **局部优先**：线程优先复用本地空闲 Slot，减少全局锁冲突。
* **无锁回收**：全局空闲链表采用原子操作，支持高并发场景。
* **批量分配**：Block 内批量切分 Slot，降低系统分配压力。
